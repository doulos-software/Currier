Framework

Dependencies:
	memorizing notes on the fretboard
		start with string 6, then progress
	it is highly advisable to facilitate the framework via ear-training

Philosophy:
	all music revolves around these horizontal and vertical patterns
	playability demands vertical movements, tonality demands horizontal movements - a balance must be struck
	slow is smooth, smooth is fast
		the primary goal of all practice is produce a solid, deliberate tone
		the secondary goal is to maintain this tone while harmonizing and making the melody
		the tertiary goal is to synthesize the music until it can flow effortlessly (i.e. quickly)
	ideally we do not memorize notes, but named patterns - learn the patterns and how to describe them using musical terminology
